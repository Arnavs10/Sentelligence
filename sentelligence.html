All 4 Spark Models — Relative Performance
Ensemble (Majority Vote): Predictions from all 4 models are combined — if ≥ 3 of 4 predict positive, the final label is positive. Implemented via .sum(axis=1) >= 3 across lr_pred, nb_pred, svm_pred, rf_pred columns.
Note: The model scores above are relative (actual values produced at runtime). All 4 evaluators used: MulticlassClassificationEvaluator with metrics accuracy, f1, weightedPrecision, and weightedRecall. Per-source evaluation (amazon / twitter / financial) is also performed for all 4 Spark models.

Phase 4

Sentiment Shock
Detection

Sudden sentiment shifts flagged using a rolling window Z-score algorithm applied to ensemble predictions aggregated over time windows.

50
WINDOW_SIZE
Records grouped per time index. time_index = index // 50
5
ROLLING_WINDOW
Lookback for rolling mean and std. min_periods=2
1.4
Z_THRESHOLD
Any point with |Z-score| exceeding 1.4 is flagged as is_shock = True
Z-score formula (from code)
z = (sentiment_avg − rolling_mean) / rolling_std
rolling_std.replace(0, np.nan).fillna(0)
is_shock = |z| > 1.4
sentiment_avg rolling_mean is_shock = True
Each source requires ≥ WINDOW_SIZE × 3 records to run shock detection. Time points with n < WINDOW_SIZE // 2 are filtered out.
Phase 5

Spark SQL
Hive-Compatible

Three Spark DataFrames registered as temp views using createOrReplaceTempView — Hive-compatible, no external Hive installation required.

model_metrics
model_name, accuracy, precision_val, recall_val, f1_score, source — for all models overall + per-source
🏷️
sentiment_predictions
row_id, source, actual_label, predicted_label — ensemble predictions per test record
⚡
shock_detection
time_index, sentiment_avg, rolling_mean, rolling_std, z_score, is_shock, source
Query 1
Overall model ranking by accuracy
SELECT model_name, accuracy, f1_score
FROM model_metrics
WHERE source = 'overall'
ORDER BY accuracy DESC
Query 2
Best model per source (correlated subquery)
SELECT source, model_name, accuracy
FROM model_metrics m1
WHERE source != 'overall'
AND accuracy = (SELECT MAX(accuracy)
  FROM model_metrics m2
  WHERE m2.source = m1.source
  AND m2.source != 'overall')
Query 3
Sentiment distribution per source
SELECT source, predicted_label AS sentiment,
  COUNT(*) AS count
FROM sentiment_predictions
GROUP BY source, predicted_label
ORDER BY source, sentiment
Queries 4 · 5 · 6
Positive %, prediction accuracy %, shock summary
SELECT source,
 ROUND(SUM(CASE WHEN predicted_label=1
  THEN 1 ELSE 0 END)*100.0/COUNT(*),2) positive_pct, ...
FROM sentiment_predictions GROUP BY source

SELECT source, SUM(is_shock=true) shocks,
 ROUND(MAX(z_score),3) max_zscore
FROM shock_detection GROUP BY source
